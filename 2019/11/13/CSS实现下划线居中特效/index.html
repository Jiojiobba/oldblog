CSS实现下划线居中特效

：afrer伪元素在元素之后添加内容
这个伪元素允许在元素内容的最后面插入生成内容。
这个伪元素默认是行内元素，不过可以使用属性 display 改变这一点。

bottom:5px;(相对.title这个盒子，向下5px)
left: 50%; （ 因为是绝对定位，所以向右移一半）
transform：translateX（-50%）(从中间向两边延伸)
